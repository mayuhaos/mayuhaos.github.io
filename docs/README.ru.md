- [x] **Astro + Tailwind CSS** - сверхбыстрая генерация статических сайтов
- [x] **Плавные анимации** - анимации переходов страниц Swup
- [x] **Адаптивный дизайн** - идеальная адаптация для десктопа, планшета и мобильных устройств

- [x] **Полнотекстовый поиск** - клиентский поиск на базе Pagefind

## 🚀 Быстрый старт

### Требования

- Node.js ≥ 22
- pnpm ≥ 9

### Локальная разработка

1. **Клонировать репозиторий:**
   ```bash
   git clone https://github.com/mayuhaos/mayuhaos.github.io.git
   cd mayuhaos.github.io
   ```

2. **Установить зависимости:**
   ```bash
   npm install -g pnpm
   pnpm install
   ```

3. **Запустить сервер разработки:**
   ```bash
   pnpm dev
   ```
   Блог будет доступен по адресу `http://localhost:4321`

## 🧞 Команды

| Команда                    | Действие                                            |
|:---------------------------|:----------------------------------------------------|
| `pnpm install`             | Установить зависимости                              |
| `pnpm dev`                 | Запустить сервер на `localhost:4321`                 |
| `pnpm build`               | Собрать сайт в `./dist/`                            |
| `pnpm preview`             | Предпросмотр собранного сайта                       |
| `pnpm new-post <filename>` | Создать новую статью                                |
