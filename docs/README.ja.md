## 🚀 クイックスタート

### 必要環境

- Node.js ≥ 22
- pnpm ≥ 9

### ローカル開発

1. **リポジトリをクローン：**
   ```bash
   git clone https://github.com/mayuhaos/mayuhaos.github.io.git
   cd mayuhaos.github.io
   ```

2. **依存関係をインストール：**
   ```bash
   npm install -g pnpm
   pnpm install
   ```

3. **開発サーバーを起動：**
   ```bash
   pnpm dev
   ```
   ブログは `http://localhost:4321` で利用可能

## 🧞 コマンド

| Command                    | Action                                              |
|:---------------------------|:----------------------------------------------------|
| `pnpm install`             | 依存関係をインストール                               |
| `pnpm dev`                 | `localhost:4321` で開発サーバーを起動        |
| `pnpm build`               | `./dist/` にサイトをビルド            |
| `pnpm preview`             | ビルド済みサイトをプレビュー        |
| `pnpm new-post <filename>` | 新しい記事を作成                                   |
